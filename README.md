# Python网上订餐管理系统 python1043

## 项目概述

本项目是基于Python语言和Oracle数据库开发的网上订餐管理系统，包含登录注册、菜品信息管理、订单处理等多个功能模块，适用于餐饮行业的日常运营管理。

## 技术栈

- **后端：** Python
- **数据库：** Oracle

## 功能模块

1. **登录注册模块**
   - 用户登录
   - 用户注册

2. **首页展示**
   - 菜品推荐
   - 最新活动

3. **菜品信息管理**
   - 菜品添加
   - 菜品编辑
   - 菜品删除

4. **订单信息管理**
   - 订单查询
   - 订单处理
   - 订单跟踪

5. **用户信息管理**
   - 用户资料编辑
   - 用户地址管理

6. **系统消息管理**
   - 消息发送
   - 消息接收

7. **积分管理**
   - 积分获取
   - 积分消费

8. **管理员信息管理**
   - 管理员登录
   - 管理员权限设置

## 系统角色

- 普通用户：进行登录、点餐、查看订单等操作
- 管理员：负责菜品管理、订单处理、用户管理和系统消息发送

## 运行环境

- Python环境
- Oracle数据库
